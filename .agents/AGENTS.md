# Custom Agent Rules

## Website Content Management System (CMS)
For all web application or website projects, always implement a visual Content Management System (CMS) or admin panel (e.g. at `/cms` or `/admin`) secured with a login gate. This system must allow non-technical administrators to visually edit, add, and delete website content, settings, parameters, and company metadata (such as company details, contact info, rules, and limits) without having to modify HTML, database, or backend files directly.
